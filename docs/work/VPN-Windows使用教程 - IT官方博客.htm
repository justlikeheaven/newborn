      VPN-Windows使用教程

      作者: 卢建楷；类别: 桌面服务

      （注：VPN是用于在外网连接到公司网络访问公司资源，在公司内网不需要拨VPN也不能使用VPN）

         

      VPN 连接有2种方法，请任选其中一种方法进行连接。

         

      方法一： 使用EasyConnect 客户端连接VPN

       1. 首次使用需要安装客户端组件，使用IE浏览器访问https://sslvpn.boyaa.com, 点击"下载安装组件"；

       

      2.  运行安装组件，等待组件安装完成。

      3. 在桌面找到EasyConnect 客户端 ;

      4. 打开客户端填入服务器地址：https://sslvpn.boyaa.com 点击"连接"；

      5. 输入您的账号和密码，点击"登录"；

      6. 在窗口右下方可以看到EasyConnect显示"已连接"；

      7. 右键EacyConnect图标，打开"系统设置"

      8. 勾选"永久在线"，保存设置。

      9.现在您可以访问到公司内部服务器资源了, 下次拨VPN直接在桌面打开EasyConnect客户端进行连接即可。

      10.使用完VPN，右键EacyConnect图标"退出"，可以断开VPN连接。

         

         

         

      方法二：使用系统自带的VPN程序连接

       1. 打开"控制面板"—"网络和共享中心", 点击"设置新的连接或网络"

      2. 选择"连接到工作区"，下一步；

       

      3. 选择"否，创建新连接"，下一步；

       

      4.  选择"使用我的Internet连接(VPN)"

      5. 填写：

              Internet 地址： sslvpn.boyaa.com   

              目标名称： BoyaaVPN   

              后点击"创建"；

       

      6. 点击桌面右下角网络图标，   

      在弹出的右侧窗口可以看到新建的BoyaaVPN， 选择BoyaaVPN点击"连接"。

      7.  输入你的账号和密码，点击"确定"；

      8.  查看VPN状态为已连接, 现在您可以访问到公司内部服务器资源了；

       

      9.  使用完VPN，点击"断开连接"，即可。
