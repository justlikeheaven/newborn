服务器速度测试
1.ping 123.57.92.9 -t
每一个被发送出的IP信息包都有一个TTL域，该域被设置为一个较高的数值（在本例中ping信息包的TTL值为255)。当信息包在网络中被传输时，TTL的域值通过一个路由器时递减1；当TTL 递减到0时，信息包被路由器抛弃。
TTL通常表示包在被丢弃前最多能经过的路由器个数。
